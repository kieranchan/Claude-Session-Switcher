# Contributing to Claude Account Switcher

First off, thank you for considering contributing to this project! Your help is greatly appreciated.

## How to Contribute

There are many ways to contribute to this project, including:

*   **Reporting Bugs:** If you find a bug, please open an issue on GitHub and provide as much detail as possible.
*   **Suggesting Features:** If you have an idea for a new feature, please open an issue on GitHub to discuss it.
*   **Submitting Pull Requests:** If you would like to contribute code, please open a pull request with your changes.

## Development Setup

To set up the development environment for this project, you will need to:

1.  **Clone the repository:**
    ```
    git clone https://github.com/your-username/claude-switcher.git
    ```
2.  **Install the dependencies:** This project has no external dependencies.
3.  **Load the extension in Chrome:**
    *   Open Chrome and navigate to `chrome://extensions`.
    *   Enable "Developer mode".
    *   Click "Load unpacked" and select the project directory.

## Code Style

This project uses a simple and clean coding style. Please try to follow the existing code style when contributing.

*   **JavaScript:** Use modern JavaScript (ES6+) and follow the existing code style.
*   **CSS:** Use modern CSS and follow the existing code style.
*   **HTML:** Use modern HTML and follow the existing code style.

## Pull Request Process

1.  **Fork the repository:** Create a fork of the repository to your own GitHub account.
2.  **Create a branch:** Create a new branch for your changes.
3.  **Make your changes:** Make your changes to the code.
4.  **Test your changes:** Test your changes to make sure they work as expected.
5.  **Commit your changes:** Commit your changes with a clear and concise commit message.
6.  **Push your changes:** Push your changes to your fork.
7.  **Open a pull request:** Open a pull request from your fork to the main repository.

Thank you for your contributions!
